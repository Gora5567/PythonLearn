1.  Суммирование
Напиши функцию add_all(*args), которая возвращает сумму всех переданных чисел.


2. Поиск максимального
Напиши функцию find_max(*args), которая возвращает максимальное значение из аргументов.


3. Подсчет количества
Напиши функцию count_args(*args), которая возвращает количество переданных аргументов.


4.  Конкатенация строк
Функция concat_strings(*args) возвращает одну строку, полученную соединением всех строковых аргументов.


5. Список квадратов
squares(*args) возвращает список квадратов всех переданных чисел.


🟡 Средние

6. Фильтрация по типу
filter_by_type(type_, *args) — возвращает список только тех аргументов, которые относятся к заданному типу.


7. Индексация
get_element(index, *args) — верни элемент из args по индексу, если индекс в диапазоне. Иначе – None.


8. Аргументы по длине
args_by_length(n, *args) — верни список только тех аргументов, длина которых больше n.


9. Преобразование типов
convert_to_str(*args) — верни список строковых представлений всех аргументов.


Математическое выражение
calculate(operator, *args) — применяет оператор ('+', '*') ко всем числам.
Пример: calculate('+', 1, 2, 3) → 6

print_info(**kwargs) — выводит каждую пару ключ-значение в формате key: value.


Подсчет значений
count_values(**kwargs) — возвращает количество переданных ключей.



Проверка ключа
has_key(key, **kwargs) — возвращает True, если ключ есть в словаре.


Слияние со словарем по умолчанию
