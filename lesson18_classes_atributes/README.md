1.) Создай класс Car с атрибутом объекта 
brand. Создай объект и выведи этот атрибут.

2.) Создай класс Book с атрибутами объекта
title и author. Создай объект и выведи оба атрибута.

3.) Создай класс Player с атрибутом объекта
score = 0. Увеличь значение score после создания объекта.

4.) Создай класс Cat с атрибутом объекта
name и методом, который выводит это имя.

5.) Создай класс Phone с атрибутом класса
manufacturer и атрибутом объекта model. Выведи оба атрибута.

6.) Создай класс Student с атрибутом класса
school = "№12" и атрибутами объекта name и grade. Создай два объекта и сравни school.

7.) Создай класс Counter с атрибутом класса
total_created = 0, который увеличивается при каждом создании объекта.

8.) Создай класс Rectangle с атрибутами width
и height и методом area(), который возвращает площадь.

9.) Создай класс User с приватным атрибутом
__password и методами set_password() и get_password().

10.) Создай класс GameCharacter с атрибутами
hp, damage, armor и методом take_damage(amount), который уменьшает здоровье с учётом брони.
